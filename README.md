# Grunt Bower Boilerplate

It's my project template :)

## Set up

1. `git clone git@github.com:FiNGAHOLiC/grunt-bower-boilerplate.git`
2. `cd grunt-bower-boilerplate`
3. `npm install`
4. `bower install`
5. `git submodule init`
6. `git submodule update`

## Grunt tasks

* `grunt development`
* `grunt production`
